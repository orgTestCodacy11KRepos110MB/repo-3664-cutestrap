Sorry about the alert!

If you'll notice, we passed an alert function to our relay. Our relay called it immediately with the most recent state.

Starting from the top, we're setting a variable called `tag`. We then set our clicked state to be 0. That's the value from the alert.

Next, we've got an event that listens for a click on our button. After a click, it sets the clicked state to be 1. Every time the button is clicked, an alert will display with 1. 

<button>Click Me</button>

<script type="module">
  import {
    set,
    on,
    relay
  } from '../js/vendor/ion/app.js?CACHEBUST'

  const tag = 'button'

  set(tag, { clicked: 0 })

  on('click', tag, ({ target }) => {
    set(tag, { clicked: 1 })
  })

  relay(state => {
    alert( JSON.stringify(state[tag]) )
  })
</script>
